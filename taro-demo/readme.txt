不使用typeScript
# npm script
$ npm run dev:weapp
$ npm run build:weapp

# 仅限全局安装
$ taro build --type weapp --watch
$ taro build --type weapp

# npx 用户也可以使用
$ npx taro build --type weapp --watch
$ npx taro build --type weapp

目录设置
/service service服务，如接口
/template 公共模块 
/utils  存放工具类 tools.js
/static  静态资源 图片 css
